# RefundType

Categorizes the reason for a refund to the frontend: - `FULL_REVERSAL`: Entire budgeted amount returned (e.g. order cancelled, declined, or no items found). - `PARTIAL_REFUND`: Refund for price differences after shopping or partial returns. 

## Enum

* `FullReversal` (value: `'FULL_REVERSAL'`)

* `PartialRefund` (value: `'PARTIAL_REFUND'`)
